# Simple Interest Calculator

This project calculates Simple Interest using the formula:

Simple Interest = (P × R × T) / 100

Where:
P = Principal amount  
R = Rate of interest  
T = Time  

## Example
If P = 1000, R = 5%, T = 2 years

Simple Interest = (1000 × 5 × 2) / 100 = 100

## Purpose
This project helps beginners understand basic programming and financial calculations.
